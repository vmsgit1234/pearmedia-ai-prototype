# Pear Media – AI Studio

A web prototype that uses AI APIs to enhance text prompts and generate images.

## Live Demo
https://vmsgit1234.github.io/pearmedia-ai-prototype

## Features

### Text Workflow
1. User types a simple image idea
2. AI enhances it into a detailed prompt
3. User reviews and edits the prompt
4. Image is generated using Pollinations AI

### Image Workflow
1. User uploads an image
2. App analyzes the style and colors
3. Generates a variation using Pollinations AI

## Tech Stack
- HTML, CSS, JavaScript
- Pollinations AI (free, no API key needed)
- Hosted on GitHub Pages

## How to Run Locally
1. Download the repository
2. Open index.html in your browser
3. No installation needed!

## APIs Used
- Pollinations AI - for image generation (free)
